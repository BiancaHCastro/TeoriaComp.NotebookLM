# NotebookLM - Impulsionando aprendizado

## **Projeto - NotebookLM produzido para impulsionar estudo**

Projeto com o objetivo de gerar um ebook de tecnologia, usando ferramentas de IA como acelerador, através de estratégias de engenharia de prompt.

Projeto com o objetivo de criar um notebook na plataforma do Google: NotebookLM, com o intuito de aumentar o desempenho de estudos e conhecimentos adquirido acerca de um determinado conteúdo acadêmico.

Neste caso em específico, os conteúdos para conclusão da disciplina de Aspectos Teóricos da Computação, que é constituída pelos seguintes assuntos:

**Formalismos computacionais para construção de linguagens.**

- **Sistemas formais: fundamentos e aplicações**
- **Gramáticas, alfabetos e linguagens**
- **A estratégia dos autômatos finitos e linguagens regulares**
- **A pilha nos autômatos para resolver problemas potencialmente recursivos**
- **Aplicação: analisador léxico e sintático.**

**Os limites da computabilidade perante os modelos formais de computação.**

- **A relação dos problemas de Hilbert com a computação: Tese de Church, Teorema de Gödel.**
- **Computação de algoritmos e equivalência com Máquinas de Turing: funções parcialmente recursivas, máquinas com registradores**
- **Os problemas clássicos de indecidibilidade: correspondência de Post, castor atarefado, problema da parada**
- **A classes de problemas e a complexidade de tempo: P, NP e NP-completo, teorema de Cook, redução de problemas**
- **Aplicação: geração de código em máquina de registradores.**

A ideia inicial é encontrar os principais pontos a serem aprendidos e as dificuldades já identificadas em sala de aula para utilizar o NotebookLM como uma ferramenta de estudo guiado. Considerando que o chat atuará como um chatbot especializado.

## **💻 Tecnologias utilizadas no projeto**

- [ChatGPT](https://chat.openai.com/)
- [NotebookLM](https://notebooklm.google.com/)
- [Notion](https://docs.google.com/presentation/u/1/)

## **✨ Como foi feito?**

- Análise aprofundada dos assuntos presentes na ementa da disciplina, de acordo com o que já foi visto em sala de aula
- Inclusão das fontes (livros, vídeos, artigos) no NotebookLM
- Filtrar as fontes localizadas pela própria ferramenta, mantendo apenas as que se aplicam
- Estudo guiado para cada etapa do aprendizado, usando o chat especializado.
